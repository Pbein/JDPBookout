JD Power PDF Downloader v1.0
================================

A simple desktop application to download PDF reports from JD Power Values Online.

HOW TO USE:
1. Double-click JDPowerDownloader.exe
2. Enter your JD Power username and password
3. Click "Browse" to select where PDFs should be saved
4. Set "Max Downloads" to how many PDFs you want (0 = all)
5. Set "Workers" to 5-7 (for faster downloads)
6. Click "Start Download"
7. Watch the progress bar and activity log
8. PDFs will appear in your chosen folder

TIPS:
- Start with a small test (3 PDFs) first
- Keep "Run browser in background" checked for speed
- The app remembers your settings for next time
- You can stop and resume downloads anytime

REQUIREMENTS:
- Windows 10/11 (64-bit)
- Internet connection
- JD Power credentials
- ~100 MB free disk space

NEED HELP?
- Check TESTING_GUIDE.md for detailed instructions
- Make sure you have a stable internet connection
- Verify your JD Power credentials are correct

The application will automatically:
- Log into JD Power Values Online
- Export the inventory list as CSV
- Download PDF reports for each vehicle
- Save PDFs with the correct reference numbers
- Show real-time progress and statistics
- Handle errors and recovery automatically

Enjoy your automated PDF downloads!
